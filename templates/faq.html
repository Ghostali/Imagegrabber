How does it get the images?

Using a a websites url, the application is able to crawl the given website for image tags and then save the links to the images and return them in a single page for you to see.

Why do I sometimes see empty boxes?

On some websites there are images the size of 1 pixel. When we get the image to display it shows up as en empty box. We will be looking at how to get rid of these images form the results.

Why does it take so long?

The reason why it takes so long is that the application has to go on to the site and gather the images. Sometimes the sites take long to respond and other times there are so many images to grab.

What does the scrolling do?

The scrolling allows you to select if we should scroll down the images to load up images which do not show unless you scroll down such as on google image.

You can choose how far to scroll down depending on how much images you wish to grab from the selected website.
